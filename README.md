# Windows 12 Web Version

**Windows 12 Web Version** is an experimental operating system simulation built for the web.  
It recreates familiar Windows elements — desktop, taskbar, Start menu, system apps, and animations — directly in the browser.

## ✨ Features
- Modern Windows‑style design with translucent panels
- Taskbar with clock, battery, and system controls
- Start menu with search bar and pinned apps
- Settings app with Personalisation and System sections
- Microsoft Edge browser window
- Desktop icon selection and folders
- Animations for opening, minimizing, and closing windows
- Update Record tab to track changes
